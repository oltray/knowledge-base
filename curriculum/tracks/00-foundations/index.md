# Track 00 — Foundations

> **Level:** Complete beginner
> **Goal:** Understand how computers work before writing a single line of code

---

## Why This Track Exists

Before learning any programming language, you need a mental model of what's
actually happening inside a computer. This track builds that foundation so
nothing in later tracks feels like magic.

---

## Modules
